MSP430 Microcontroller Games, WPI

January 2023 - March 2023


Simon Says:

I programmed a “Simon Says” game on an MSP430F5529 microcontroller using switch cases and pointers. The game utilizes digital I/O by reading inputs from the keypad and setting the LEDs and buzzer as outputs. The MSP430 generates a random LED sequence for each level, and as the user wins rounds, the number of levels increases and the game goes faster until it reaches the 32nd level, which is the maximum.


Attached is a video of me playing the game I created.




Guitar Hero:

I programmed a simplified version of the Guitar Hero video game on an MSP430F5529 microcontroller using structs, switch cases, and pointers. First, each note of the song “Twinkle Twinkle Little Star” was assigned one of four LED values and ceramic buzzer frequencies. Then, my function allows for the MSP430 to play each note of the song while flashing an LED and playing the buzzer, and it expects player input on one of four buttons. If the button that corresponds to the LED is pressed, the song continues until the last note and the player wins. Otherwise, they lose. This project utilized digital I/O and incorporated timers and interrupts.


Attached is a video of me playing the game I created.




Relevant skills: C & MSP430F5529 microcontroller
